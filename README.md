# AI 计算器 Web 应用

一个现代化的计算器 Web 应用，支持基础四则运算和单位转换功能。

## ✨ 功能特性

### 🧮 计算器模式

#### 标准模式
- 支持加法、减法、乘法、除法四则运算
- 小数点计算
- 清除和退格功能
- 实时显示计算结果
- 错误处理机制
- **智能输入行为**：计算后输入数字自动清空，输入运算符继续运算
- **历史记录**：显示最近3次计算的表达式和结果，自动滚动到最新记录

#### 科学模式（新增！）
- **三角函数**：sin, cos, tan（支持角度/弧度切换）
- **对数函数**：log（常用对数）, ln（自然对数）
- **幂运算**：x², x³, xʸ, eˣ, 10ˣ
- **根号运算**：平方根 √
- **数学常数**：π（圆周率）, e（自然常数）
- **括号支持**：改变运算优先级
- **阶乘运算**：n!
- **其他功能**：绝对值 abs, 倒数 1/x, 正负切换 ±

### 🔄 单位转换模式
- **长度转换**：米、千米、厘米、毫米、英尺、英寸、英里
- **重量转换**：千克、克、毫克、磅、盎司
- **温度转换**：摄氏度、华氏度、开尔文
- **面积转换**：平方米、平方千米、平方英尺、英亩、公顷

## 🚀 使用指南

### 计算器模式

#### 标准模式
1. 点击数字按钮输入数值
2. 点击运算符按钮选择运算类型
3. 点击 `=` 按钮获取结果
4. 点击 `C` 按钮清除，或 `⌫` 退格
5. **智能行为**：计算完成后
   - 输入数字 → 自动清空，开始新计算
   - 输入运算符 → 继续使用当前结果

#### 科学模式
1. 点击顶部 **“科学”** 按钮切换到科学模式
2. 选择角度模式（DEG/RAD）
3. 使用科学函数按钮进行计算
4. 支持括号、常数、幂运算等高级功能
5. 查看详细使用说明：[docs/scientific-calculator.md](./docs/scientific-calculator.md)

### 单位转换模式
1. 点击顶部 "🔄 单位转换" 切换到转换模式
2. 选择转换类型（长度/重量/温度/面积）
3. 在 "从" 区域输入数值并选择起始单位
4. 在 "到" 区域选择目标单位
5. 点击 "转换" 按钮查看结果
