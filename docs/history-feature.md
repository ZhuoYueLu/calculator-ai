# 计算历史记录功能说明

## 🎯 功能概述

计算器现在支持**历史记录显示**，可以在主显示屏上方看到最近3次计算的表达式和结果。

```
┌─────────────────────────────┐

│   [标准] [科学]              │
├─────────────────────────────┤
│   [DEG] [RAD]               │
├─────────────────────────────┤
│  5 + 3                      │ ← 历史记录1（较早）
│  = 8                        │
├─────────────────────────────┤
│  10 × 5                     │ ← 历史记录2
│  = 50                       │

│  = 0.5                      │
├═════════════════════════════┤ ← 分隔线
│                         42  │ ← 当前输入/结果
├─────────────────────────────┤
│ 按钮区域...                  │
└─────────────────────────────┘
```

## ✨ 功能特点

### 1. 自动记录
- 每次按下 `=` 计算完成后，自动添加到历史记录
- 无需手动操作

### 2. 显示内容
每条历史记录包含：
- **表达式**：输入的算式（灰色小字）
- **结果**：计算结果（紫色大字）

### 3. 保留数量
- 最多保留**最近3条**记录
- 新记录会自动挤掉最旧的记录

### 4. 自动清空
- 点击 `C` 清除按钮时，历史记录也会清空

## 📊 技术实现

### 状态管理
```javascript
const [history, setHistory] = useState([]);
```

**数据结构**:
```javascript
[
  { expression: "5 + 3", result: "8" },
  { expression: "10 × 5", result: "50" },
  { expression: "sin(30)", result: "0.5" }
]
```

### 添加记录
```javascript
// 计算完成后
setHistory(prev => {
  const newHistory = [...prev, { 
    expression: input, 
    result: resultStr 
  }];
  // 只保留最近3条
  return newHistory.slice(-3);
});
```

### 清空记录
```javascript
// 点击 C 按钮时
if (val === 'C') {
  setInput('');
  setHistory([]); // 清空历史
  setIsNewInput(true);
  return;
}
```

### JSX 渲染
```jsx
{history.length > 0 && (
  <div className="history-display">
    {history.map((item, index) => (
      <div key={index} className="history-item">
        <div className="history-expression">{item.expression}</div>
        <div className="history-result">= {item.result}</div>
      </div>
    ))}
    <div className="history-divider"></div>
  </div>
)}
```
