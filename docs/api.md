# API 文档

## 概述

本文档描述 AI 计算器应用的功能接口和使用方法。

## 计算器功能

### 基本运算

#### 加法
- **操作**: 输入数字 → 点击 `+` → 输入数字 → 点击 `=`
- **示例**: `5 + 3 = 8`

#### 减法
- **操作**: 输入数字 → 点击 `-` → 输入数字 → 点击 `=`
- **示例**: `10 - 4 = 6`

#### 乘法
- **操作**: 输入数字 → 点击 `*` → 输入数字 → 点击 `=`
- **示例**: `3 * 7 = 21`

#### 除法
- **操作**: 输入数字 → 点击 `/` → 输入数字 → 点击 `=`
- **示例**: `15 / 3 = 5`

### 特殊功能

#### 清除 (C)
- **功能**: 清空当前输入
- **操作**: 点击 `C` 按钮

#### 小数点
- **功能**: 输入小数
- **示例**: `3.14 + 2.86 = 6`

#### 连续运算
- **功能**: 支持连续进行多次运算
- **示例**: `5 + 3 * 2 = 16`（遵循运算优先级）

### 错误处理

| 错误类型 | 显示 | 说明 |
|---------|------|------|
| 语法错误 | Error | 表达式格式不正确 |
| 除以零 | Infinity | 数学上的无穷大 |
| 无效输入 | Error | 包含非数字字符 |

## 单位转换功能

### 长度转换

#### 支持单位

| 单位代码 | 单位名称 | 相对于米的比率 |
|---------|---------|--------------|
| m | 米 | 1 |
| km | 千米 | 0.001 |
| cm | 厘米 | 100 |
| mm | 毫米 | 1000 |
| ft | 英尺 | 3.28084 |
| in | 英寸 | 39.3701 |
| mi | 英里 | 0.000621371 |

#### 转换公式
```
目标值 = (输入值 / 起始单位比率) × 目标单位比率
```

#### 示例
- 100 米 = 0.1 千米
- 1 英尺 = 30.48 厘米
- 1 英里 = 1.60934 千米

### 重量转换

#### 支持单位

| 单位代码 | 单位名称 | 相对于千克的比率 |
|---------|---------|----------------|
| kg | 千克 | 1 |
| g | 克 | 1000 |
| mg | 毫克 | 1000000 |
| lb | 磅 | 2.20462 |
| oz | 盎司 | 35.274 |

#### 示例
- 1 千克 = 1000 克
- 1 磅 = 0.453592 千克
- 1 盎司 = 28.3495 克

### 温度转换

#### 支持单位

| 单位代码 | 单位名称 |
|---------|---------|
| c | 摄氏度 |
| f | 华氏度 |
| k | 开尔文 |

#### 转换公式

**摄氏转华氏**
```
F = C × 9/5 + 32
```

**华氏转摄氏**
```
C = (F - 32) × 5/9
```

**摄氏转开尔文**
```
K = C + 273.15
```

**开尔文转摄氏**
```
C = K - 273.15
```

**华氏转开尔文**
```
K = (F - 32) × 5/9 + 273.15
```

**开尔文转华氏**
```
F = (K - 273.15) × 9/5 + 32
```

#### 示例
- 0°C = 32°F = 273.15K
- 100°C = 212°F = 373.15K
- -40°C = -40°F

### 面积转换

#### 支持单位

| 单位代码 | 单位名称 | 相对于平方米的比率 |
|---------|---------|------------------|
| sqm | 平方米 | 1 |
| sqkm | 平方千米 | 0.000001 |
| sqft | 平方英尺 | 10.7639 |
| acre | 英亩 | 0.000247105 |
| hectare | 公顷 | 0.0001 |

#### 示例
- 1 平方千米 = 1,000,000 平方米
- 1 英亩 = 4046.86 平方米
- 1 公顷 = 10,000 平方米

## 使用流程

### 计算器模式

```
1. 启动应用 → 默认进入计算器模式
2. 点击数字按钮输入第一个数
3. 点击运算符按钮 (+, -, *, /)
4. 点击数字按钮输入第二个数
5. 点击 = 按钮查看结果
6. 如需重新计算，点击 C 按钮清除
```

### 单位转换模式

```
1. 点击顶部 "🔄 单位转换" 按钮
2. 选择转换类型（长度/重量/温度/面积）
3. 在 "从" 区域：
   - 输入数值
   - 选择起始单位
4. 在 "到" 区域：
   - 选择目标单位
5. 点击 "转换" 按钮
6. 查看结果显示
```

## 注意事项

### 精度问题
- 浮点数运算可能存在微小误差
- 结果保留4位小数
- 自动去除末尾的零

### 输入限制
- 只接受数字和小数点
- 不支持科学计数法
- 不支持括号和复杂表达式

### 边界情况
- 除以零返回 Infinity
- 超大数字可能显示为科学计数法
- 负数支持正常运算

## 故障排除

### 常见问题

**Q: 计算结果不准确？**
A: 这是浮点数精度的正常现象，结果已保留4位小数。

**Q: 单位转换结果为零？**
A: 检查是否输入了有效数字，确认单位选择正确。

**Q: 页面加载缓慢？**
A: 清除浏览器缓存，或使用无痕模式访问。

**Q: 按钮点击无反应？**
A: 检查 JavaScript 是否被禁用，或尝试刷新页面。
