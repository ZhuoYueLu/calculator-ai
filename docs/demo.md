### 复杂表达式

#### 示例 15: 综合计算
```
表达式: sin(30°) + cos(60°) × 2²

操作步骤:
1. sin→(→3→0→)→+
2. cos→(→6→0→)→*
3. 2→x²→=

结果: 2.5

验证:
sin(30°) = 0.5
cos(60°) = 0.5
2² = 4
0.5 + 0.5 × 4 = 0.5 + 2 = 2.5 ✓
```

结果: 125 J
```

#### 示例 17: 数学恒等式

```

## 💡 实用技巧

### 技巧 1: 快速平方
```
计算 7²
方法 1: 7→xʸ→2→=
方法 2: 7→x²→=  ← 更快！
```

### 技巧 2: 使用括号
```
计算 (3+2)×(4-1)
正确: (→3→+→2→)→*→(→4→-→1→)→=
错误: 3→+→2→*→4→-→1→=  ← 优先级不同
```

### 技巧 3: 连续计算
```
计算 sin(30) + cos(60)
步骤:
1. sin→(→3→0→)→+
2. cos→(→6→0→)→=
无需中间按等号
```

### 技巧 4: 利用常数
```
计算圆面积 A = πr² (r=5)
输入: π * 5^2
操作: π→*→5→x²→=
结果: 78.539816...
```

### 技巧 5: 角度转换
```
将 180° 转为弧度
公式: 弧度 = 角度 × π/180
输入: 180 * π / 180
结果: π ≈ 3.14159...
```

## ⚠️ 常见错误

### 错误 1: 忘记括号
```
❌ sin 30 → 语法错误
✅ sin(30) → 正确
```

### 错误 2: 角度模式混淆
```
在 RAD 模式下计算 sin(90)
❌ sin(90) ≠ 1  （因为 90 弧度 ≠ 90°）
✅ 切换到 DEG 模式，或输入 sin(π/2)
```

### 错误 3: 除以零
```
❌ 1 / 0 → Error 或 Infinity
✅ 避免除以零的操作
```

### 错误 4: 负数开方
```
❌ √(-4) → Error（实数范围内无解）
✅ 确保根号内为非负数
```
