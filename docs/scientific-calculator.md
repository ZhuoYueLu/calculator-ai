# 科学计算器使用指南

## 🎯 功能概述

AI 计算器现已升级为**专业科学计算器**，支持标准模式和科学模式两种模式。

## 📱 模式切换

### 标准模式（基础四则运算）
- 加法、减法、乘法、除法
- 小数点运算
- 清除和退格功能

### 科学模式（高级数学运算）
点击顶部的 **"科学"** 按钮切换到科学模式，解锁更多功能。

## 🔬 科学计算器功能

### 1. 三角函数

| 按钮 | 功能 | 说明 |
|------|------|------|
| sin | 正弦函数 | sin(x) |
| cos | 余弦函数 | cos(x) |
| tan | 正切函数 | tan(x) |

**角度/弧度切换**：
- **DEG**：角度模式（默认）
- **RAD**：弧度模式

示例：
```
sin(30) = 0.5 （角度模式）
sin(π/6) = 0.5 （弧度模式）
```

### 2. 对数函数

| 按钮 | 功能 | 说明 |
|------|------|------|
| log | 常用对数 | log₁₀(x) |
| ln | 自然对数 | ln(x) = logₑ(x) |

示例：
```
log(100) = 2
ln(e) = 1
```

### 3. 幂运算

| 按钮 | 功能 | 说明 |
|------|------|------|
| x² | 平方 | x^2 |
| x³ | 立方 | x^3 |
| xʸ | 任意次幂 | x^y |
| eˣ | e的x次方 | e^x |
| 10ˣ | 10的x次方 | 10^x |

示例：
```
5 x² = 25
2 xʸ 10 = 1024
eˣ 1 = 2.71828...
```

### 4. 根号运算

| 按钮 | 功能 | 说明 |
|------|------|------|
| √ | 平方根 | √x |

示例：
```
√9 = 3
√2 = 1.41421...
```

### 5. 数学常数

| 按钮 | 常数 | 值 |
|------|------|-----|
| π | 圆周率 | 3.14159... |
| e | 自然常数 | 2.71828... |

### 6. 其他功能

| 按钮 | 功能 | 说明 |
|------|------|------|
| ( ) | 括号 | 改变运算优先级 |
| n! | 阶乘 | n! = 1×2×3×...×n |
| 1/x | 倒数 | 1/x |
| abs | 绝对值 | \|x\| |
| ± | 正负切换 | 改变符号 |
| ⌫ | 退格 | 删除最后一个字符 |
| C | 清除 | 清空所有输入 |

## 🎨 按钮颜色说明

- **紫色按钮**：三角函数和对数函数（sin, cos, tan, log, ln）
- **青色按钮**：数学常数（π, e）
- **粉色按钮**：幂运算（x², x³, xʸ, eˣ, 10ˣ）
- **橙色按钮**：退格键（⌫）
- **红色按钮**：清除键（C）
- **黄色按钮**：运算符（+, -, *, /）
- **绿色按钮**：等于号（=）
- **白色按钮**：数字和其他功能

## ⚙️ 角度/弧度模式

### DEG（角度模式）
- 三角函数使用角度作为输入
- 例如：sin(90) = 1

### RAD（弧度模式）
- 三角函数使用弧度作为输入
- 例如：sin(π/2) = 1

**切换方法**：点击顶部的 DEG 或 RAD 按钮

## 🔢 运算优先级

计算器遵循标准数学运算优先级：
1. 括号 ()
2. 函数（sin, cos, log 等）
3. 幂运算 (^)
4. 乘除 (*, /)
5. 加减 (+, -)

示例：
```
2 + 3 × 4 = 14 （不是 20）
(2 + 3) × 4 = 20
```

## ⚠️ 注意事项

### 1. 阶乘限制
- 只支持非负整数
- 过大的数可能导致溢出
- 示例：5! = 120, 10! = 3628800

### 2. 定义域限制
- log(x)：x 必须 > 0
- ln(x)：x 必须 > 0
- √x：x 必须 ≥ 0
- tan(90°)：无定义

### 3. 精度问题
- 浮点数运算可能有微小误差
- 结果保留10位有效数字
- 示例：0.1 + 0.2 = 0.30000000000000004

### 4. 错误处理
如果输入无效表达式，会显示 "Error"
- 点击 C 清除后重新输入

## 💻 技术实现

### 表达式解析
- 使用 JavaScript `Function` 构造函数
- 自动转换数学符号为标准语法
- 支持角度/弧度自动转换

### 特殊处理
```javascript
// 角度转弧度（DEG模式）
sin(30) → Math.sin(Math.PI/180*30)

// 数学常数替换
π → Math.PI
e → Math.E

// 函数映射
log → Math.log10
ln → Math.log
sqrt → Math.sqrt
```
