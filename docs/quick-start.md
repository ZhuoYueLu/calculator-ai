# 快速开始指南

## 🚀 5分钟快速上手

### 前置检查
确保已安装：
- ✅ Node.js (>= 16)
- ✅ Pnpm

### 第一步：安装依赖
```bash
pnpm install
```

### 第二步：启动开发服务器
```bash
pnpm dev
```

### 第三步：访问应用
浏览器打开：http://localhost:5173（或终端显示的端口）

## 📱 使用指南

### 计算器模式（默认）

#### 基本运算
1. 点击数字按钮输入第一个数
2. 点击运算符（+、-、*、/）
3. 点击数字按钮输入第二个数
4. 点击 `=` 查看结果

#### 示例
```
计算 15 + 27：
1. 点击 1 → 5
2. 点击 +
3. 点击 2 → 7
4. 点击 =
结果：42
```

#### 清除操作
点击 `C` 按钮清空当前输入

### 单位转换模式

#### 切换模式
点击顶部的 "🔄 单位转换" 按钮

#### 长度转换示例
将 100 米转换为千米：
1. 选择 "长度" 类型
2. 输入框输入：100
3. "从" 选择：米
4. "到" 选择：千米
5. 点击 "转换"
结果：0.1

#### 温度转换示例
将 100 摄氏度转换为华氏度：
1. 选择 "温度" 类型
2. 输入框输入：100
3. "从" 选择：摄氏度
4. "到" 选择：华氏度
5. 点击 "转换"
结果：212

## 💡 使用技巧

### 计算器
1. **连续运算**: 可以直接输入 `5 + 3 * 2 =`，会按照数学优先级计算
2. **小数运算**: 支持小数点，如 `3.14 + 2.86`
3. **错误恢复**: 如果显示 Error，点击 C 重新开始

### 单位转换
1. **快速切换**: 切换类型时会自动重置单位和数值
2. **精度控制**: 结果保留4位小数，自动去除末尾零
3. **常用转换**: 
   - 1 米 = 100 厘米
   - 1 千克 = 1000 克
   - 0°C = 32°F

## 🔧 常见问题

### Q: 页面打不开？
**A**: 检查终端是否有报错，尝试：
```bash
# 停止服务器（Ctrl+C）
# 重新启动
pnpm dev
```

### Q: 计算结果不对？
**A**: 
- 检查输入是否正确
- 注意运算优先级
- 浮点数可能有微小误差

### Q: 转换结果为 0？
**A**: 
- 确认输入了有效数字
- 检查单位选择是否正确
- 点击转换按钮

### Q: 界面显示异常？
**A**: 
- 刷新浏览器页面
- 清除浏览器缓存
- 检查是否禁用了 JavaScript
